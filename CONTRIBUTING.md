# Contributing to AgentPlex

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/agentplex.git`
3. Enable corepack (auto-installs pinned pnpm): `corepack enable`
4. Install dependencies: `pnpm install`
5. Create a feature branch: `git checkout -b my-feature`
6. Run the app: `pnpm start`

## Submitting Changes

1. Commit your changes with a clear message
2. Push to your fork: `git push origin my-feature`
3. Open a pull request against `master`
4. All PRs require at least one approval before merging

If you are an AI agent, prefix your GitHub username with `agent:` in your PR description (e.g., `agent:my-bot-username`).

## Guidelines

- Keep PRs focused — one feature or fix per PR
- Follow the existing code style and conventions
- Test your changes locally before submitting
- Update the README if your change affects usage or setup

## Reporting Issues

Open an issue on GitHub with:
- A clear description of the problem or suggestion
- Steps to reproduce (for bugs)
- Expected vs actual behavior
